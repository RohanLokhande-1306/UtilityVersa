# 🚀 UtilityVerse

**UtilityVerse** is an all-in-one web-based utility platform designed to bring useful everyday tools together in one convenient place.

The goal of UtilityVerse is to reduce the need to visit multiple websites for different everyday tasks by providing practical utilities through a single, simple, and user-friendly platform.

---

## ✨ Features

### 🧮 Smart Calculator

Perform everyday mathematical calculations through a clean and easy-to-use calculator interface.

### 🎓 Student Tools

Academic calculation tools designed for students:

* SGPA Calculator
* CGPA Calculator
* Grade-based calculations

### 💰 Finance Tools

Useful financial calculation tools:

* EMI Calculator
* GST Calculator

### 📏 Unit Converter

Quickly convert between commonly used measurement units.

### 🎂 Age Calculator

Calculate age accurately based on a selected date of birth.

### 🔐 Password Generator

Generate strong and customizable passwords.

### 📱 QR Code Generator

Create QR codes quickly for supported information and content.

# 🤖 AI Voice Customer Support Agent

UtilityVerse includes a functional **AI-powered Voice Customer Support Agent** designed to provide users with natural and interactive assistance.

The voice agent was created using **ElevenLabs Conversational AI** and configured specifically for the UtilityVerse platform.

### AI Agent Capabilities

* Handles natural voice conversations
* Answers questions about UtilityVerse
* Explains available tools and features
* Recommends suitable tools based on user requirements
* Guides users through platform features
* Handles common customer queries
* Provides automated voice-based customer support
* Maintains a professional and conversational communication style

This integration adds a conversational AI experience to UtilityVerse, allowing users to interact with an intelligent voice assistant alongside the traditional web interface.

## 🎯 Project Objective

UtilityVerse was created to solve a simple problem: users often need to visit multiple websites for different everyday tasks.

Instead of building another single-purpose calculator, UtilityVerse brings multiple practical tools together into one centralized platform.

The project also explores how modern AI technologies can improve traditional web applications through conversational and voice-based user assistance.

---

## 🤖 AI Integration

One of the key additions to UtilityVerse is its AI Voice Agent.

The agent was designed and configured specifically around UtilityVerse's features and use cases. It can communicate naturally with users, understand their questions, explain platform functionality, and provide appropriate guidance.
